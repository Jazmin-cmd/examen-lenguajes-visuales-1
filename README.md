# examen-lenguajes-visuales-1
Repositorio del examen de lengujaes visuales 1 - JAZMIN ZORRILLA
## Flujo del programa

1. Se registra el usuario.
2. Se ingresa sesión con ese usuario.
3. Se muestra una vista con todos los vendedores.
4. Permite:
   - Agregar un nuevo vendedor
   - Editar un vendedor existente
   - Eliminar un vendedor
5. Al agregar un nuevo vendedor, se guarda en la tabla `Vendedores` y recién ahí, haciendo clic en **Editar**, se puede agregar una nueva venta.
